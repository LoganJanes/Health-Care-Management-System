# Health-Care-Management-System
Web Application for Health Care Management using scikit-learn machine learning to predict if a user is at risk of heart disease using Next.js React and Tailwind.

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS, Javascript/Typescript
- **Backend**: Flask
